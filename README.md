# Nicole Solfest Photography Website

A simple, responsive static website for a photography business.

## Files

- `index.html` — homepage with hero, portfolio, services, about, and contact sections
- `styles.css` — modern dark-theme styling optimized for photography portfolios
- `script.js` — smooth scrolling for anchor links

## Getting Started

Open `index.html` in a browser to preview the site.

## Customize

- Replace the hero text and brand name with your own.
- Add your photography images to the portfolio and about sections.
- Update the contact form action if you want form submissions to send to a backend service.
